# Meals

Meals is a food menu based Mobile Application, build using flutter, for both iOS and Android. It dynamically creates categories for you, that will contain those specific meals! Further more, you can add meals you like to Favourites, so You can easily view them, anytime from the favourites screen. You can also apply filters, so meals you are not interested in, are thrown away.
