# Meals
## Meals Data
Meals is a Mobile App, that supports iOS and Android. It displays categories on Home Screen in a Grid View style. Upon opening a specific category, it will show all the meals from that category in a beautiful Card Style design. Further details of the meal can be obtained by opening the meal.
<br><br>

## Filters
Specific filters can be applied to narrow down all meals on home screen from the drawer.
<br><br>

## Favourites
There is a specific screen for favourite meals! Any meal can be added to favourite by clicking the Heart Icon in the meal details screen.
<br><br>
